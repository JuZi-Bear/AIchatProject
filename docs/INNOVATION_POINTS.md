# v2-only Innovation Points

## 1. 可观察的 AI 工作流

通过 `workflow_events`、Java RunEvent、SSE 和 Replay，把 Agent 执行过程从黑盒变成可观察事件流。

## 2. CodeAgent 安全文件操作

简化版 CodeAgent 支持受控 `read_file`、`write_file`、`list_files`，配合白名单、阻断路径、读取长度限制和 JSONL 审计。

## 3. Figma 风 Workflow Editor

Vue 前端提供无限画布、浮动 Palette、属性浮层和分支节点，适合比赛现场展示“可编排”的平台感。

## 4. Java + MySQL 平台层

Java 不只是代理，还承担任务记录、事件、模板、配置和报告索引管理。

## 5. 可回放演示闭环

拖拽 -> 执行 -> 文件生成/阻断 -> SSE -> 审计 -> Replay，形成完整比赛演示路径。
