# v2-only Demo Script

## 开场

这个项目展示的是一个可观察、可审计、可回放的 AI 多 Agent 工作流平台。

## 操作

1. 打开 Dashboard，说明平台运行状态和最近事件。
2. 打开 Workflow Editor，说明无限画布、节点 Palette 和属性浮层。
3. 拖入 CodeAgent，执行允许路径内的文件操作。
4. 展示阻断路径测试，说明安全边界。
5. 回到 RunConsole，展示 SSE 实时事件。
6. 打开审计日志和 diff。
7. 打开 Replay，按时间线回放事件。

## 总结

核心价值不是只生成代码，而是把 AI 工作流变成可管理的平台过程：有事件、有审计、有回放、有模板、有可视化。
