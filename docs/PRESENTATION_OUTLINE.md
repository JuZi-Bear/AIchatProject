# v2-only Presentation Outline

## 1. 封面

AI 多 Agent 工作流平台。

## 2. 项目价值

从生成结果展示升级为可观察、可审计、可回放的 AI 工作流平台。

## 3. 架构

Vue -> Java -> FastAPI -> LangGraph -> MySQL。

## 4. 核心能力

- Workflow Editor。
- CodeAgent 安全文件操作。
- RunEvent + SSE。
- Replay 回放。
- 模板与 Agent 注册中心。

## 5. Demo

拖拽节点 -> 执行 -> 文件生成/阻断 -> 审计日志 -> SSE -> Replay。

## 6. 成果

- v2 服务可 Docker 启动。
- 前端构建通过。
- Java 打包通过。
- CodeAgent 和 Workflow 模板 smoke 可运行。

## 7. 下一步

任务队列、权限、团队协作、更强沙箱。
