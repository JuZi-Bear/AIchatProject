## 当前重复项处理建议

- 启动说明重复：以 `README.md` 为入口，详细启动以 `OPERATION_GUIDE.md` 和 `DOCKER_COMPOSE_GUIDE.md` 为准。
- Docker 说明重复：以 `DOCKER_COMPOSE_GUIDE.md` 为主，README 只保留命令和访问地址。
- API 说明重复：以 `API_CONTRACT.md` 为主，不在其他文档重复完整请求响应。
- 技术栈说明重复：以 `TECH_STACK.md` 为主，答辩文档只保留口径化说明。
- 目录结构重复：以 `PROJECT_DIRECTORY_GUIDE.md` 为主，README 只保留精简树。
